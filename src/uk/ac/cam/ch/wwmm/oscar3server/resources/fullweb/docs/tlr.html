Token Level Regexes (TLRs)

This module assists in the recognition of chemical named entities, when the MEMM is not being used.

TLR data data

The data for the TLRs is kept in the resource file uk/ac/cam/ch/wwmm/oscar3/recogniser/tlrs.xml. This file is an XML file, with a similar structure to the Data Parser's data. Note that these regexes are processed using Java's built-in regex capabilities and not JRegex, for performance reasons. The file contains the following types of information:

tlrs
Token Level Regexes: this element contains top-level definitions of regular expressions with which to classify tokens (see the name recognition documentation for details). The tlr elements do not directly define the regexes: instead, they incorporate them by reference.
def
These define regular expressions or parts of regular expressions, which can be used by tlr elements, or be incorporated by reference into regex sections of node elements, or into def elements.
